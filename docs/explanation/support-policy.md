# Support Policy

Current support target:

- `>= 0.1.8`

Upgrade guidance:

1. Run `envlock self-update --yes` when your version is below target.
2. If self-update is unavailable in your environment, reinstall using `scripts/install.sh`.

Recommended baseline for issue reports:

- Latest stable release.
- Reproducible profile snippet.
- Full command line and output mode.
